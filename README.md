# ROS for the Alphabot2

Welcome to the "ROS for the Alphabot2" repository!

This project is designed to assist students and enthusiasts in setting up, configuring, and programming the Alphabot2 using the [Robot Operating System (ROS)](https://www.ros.org/). This repository is particularly curated for the [Autonomous Systems curricular unit](https://fenix.tecnico.ulisboa.pt/disciplinas/SAut23/2023-2024/2-semestre/objectivos) at [Instituto Superior TÃ©cnico (IST)](https://tecnico.ulisboa.pt/en/), which aims to provide a hands-on approach to learning robotics.

## Repository Contents

This repository contains:

- Source code for driver nodes
- Configuration files
- Documentation

For detailed information on how to use the resources in this repository, please visit the [wiki section](https://github.com/Kons-5/ROS-for-the-Alphabot2/wiki).
